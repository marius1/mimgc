Monochrome image converter

This tool allows you to convert a monochrome (2-bit) image to an byte array. This is useful if you use the drawBitmap function in the Adafruit GFX Libray. This tool was created to help me with the Nokia 5110/3310 lcd but any monochrome lcd should work.
